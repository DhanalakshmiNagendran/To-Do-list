# To-Do-list
simple To-Do list
A simple and interactive **To-Do List application** built using **HTML, CSS, and JavaScript**.  
This project allows users to add, mark, and remove tasks while storing data in **localStorage** so tasks remain saved even after refreshing the page.

---

## 🚀 Features

- Add new tasks
- Mark tasks as completed (single click)
- Delete tasks (double click)
- Tasks are saved using **Local Storage**
- Clean and responsive user interface
- Gradient background design

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Browser Local Storage

## ⚙️ How It Works

1. User types a task in the input field.
2. Click the **Add** button to add the task.
3. The task appears in the list.
4. Click once on a task to mark it as **completed**.
5. Double-click a task to **delete** it.
6. Tasks are automatically saved in **localStorage**.
---
## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in any web browser
